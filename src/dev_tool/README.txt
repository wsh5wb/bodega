In order to compile the file, run the command make. Then to run the executable, run './build/DevTool'
Note: You need gcc8

How to Build a Level:
First, you can load an exisiting scene using the button 'Load Scene' in the top left corner.

Moving Camera:
	W - Pan Up
	A - Pan Left
	S - Pan Down
	D - Pan Right
	Z - Scale
